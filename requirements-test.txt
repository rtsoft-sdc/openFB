# Требования для тестирования функциональных блоков OpenFB

## Основные зависимости

```
pytest>=7.0.0
pytest-cov>=3.0.0
pytest-timeout>=2.1.0
```

## Дополнительные зависимости

```
pytest-xdist>=2.5.0      # Для параллельного выполнения тестов
pytest-html>=3.1.0       # Для HTML отчетов
pytest-mock>=3.6.0       # Для мокирования
```

## Установка

```bash
# Основные зависимости
pip install pytest pytest-cov pytest-timeout

# Все зависимости для разработки
pip install -r requirements-test.txt
```

## Python версия

- Требуется: Python 3.7+
- Рекомендуется: Python 3.9+

## Проверка установки

```bash
pytest --version
```

Должна вывести версию pytest >= 7.0.0
