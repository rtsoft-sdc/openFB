Тесты OpenFB
тесты проверяют корректность работы функциональных блоков (`schedule`, обработка событий, типы данных, граничные случаи).
`openfb/resources/function_blocks_tests/`

из корня проекта (`openfb`) после установки dev-зависимостей:

Запуск
```bash
pytest -q openfb/resources/function_blocks_tests
```
