# Функциональные блоки OpenFB

## Convert

| Блок | Описание | Реализация в OpenFB |
|------|----------|---------------------|
| ARRAY2ARRAY_2_LREAL | Копирование массива LREAL[2] со входа на выход | + |
| ARRAY2VALUES_2_LREAL | Разделение массива LREAL[2] на два отдельных значения OUT_1 и OUT_2 | + |
| GET_AT_INDEX | Получить значение из массива по заданному индексу. Тип данных массива должен соответствовать типу выходных данных. | + |
| GET_STRUCT_VALUE | Извлечение значения поля структуры по имени. Поддержка вложенных полей через точку (.). Возвращает QO=TRUE при успехе | + |
| SET_AT_INDEX | Установить значение элемента массива по заданному индексу. Тип данных устанавливаемого значения должен соответствовать типу данных массива. Тип данных выходного массива должен соответствовать типу входного массива. | + |
| SET_STRUCT_VALUE | Установка значения поля структуры по имени. Поддержка вложенных полей через точку (.). Возвращает обновленную структуру | + |
| STRUCT_DEMUX | Демультиплексирование структуры - разделение полей структуры на отдельные выходы | + |
| STRUCT_MUX | Мультиплексирование отдельных входов в единую структуру | + |
| VALUES2ARRAY_2_LREAL | Объединение двух отдельных значений LREAL (IN_1, IN_2) в массив LREAL[2] | + |
